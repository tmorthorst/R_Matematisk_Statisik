# Matematisk Statistik

Dette repository indeholder R-scripts, ugeopgaver og eksamensforberedelse til faget Matematisk Statistik på HA(mat). Repositoriet fungerer som et digitalt note- og kodebibliotek over forårets pensum.

## Pensumoversigt og Ugeplan

Nedenstående oversigt viser de centrale statistiske og matematiske emner, der er dækket og kodet i repositoriet, opdelt efter undervisningsforløbet:

* **Uge 1: Deskriptiv statistik**
  Introduktion til faget og grundlæggende deskriptive metoder til dataopsummering.
* **Uge 2: Hypotesetest (Introduktion)**
  Opbygning af statistiske hypoteser, signifikansniveauer ($\alpha$) og testmetodik.
* **Uge 3: Stikprøvefordelinger og Bootstrap**
  Analyse af stikprøvefordelinger samt introduktion til non-parametrisk bootstrapping.
* **Uge 4 & 5: Estimation og Bootstrap**
  Maximum Likelihood Estimation (MLE), estimatorers egenskaber samt viderebygning på bootstrap-metoden.
* **Uge 6: Konfidensintervaller**
  Konstruktion af konfidensintervaller ved brug af pivoter, andele og bootstrap-metoder.
* **Uge 7: Hypotesetest for populationer**
  Tests for middelværdi og andele i både én og to populationer samt håndtering af parret data.
* **Uge 8: Likelihood Ratio Test**
  Praktisk anvendelse af statistiske tests og gennemgang af Likelihood ratio test.
* **Uge 9 & 10: Simpel Lineær Regression**
  Beregning af parametre, residualanalyse og hypotesetest i simple regressionsmodeller.
* **Uge 11: Multipel Lineær Regression**
  Udvidelse af regressionsmodellen til flere variable, herunder fortolkning af $R^2$ og partielle effekter.
* **Uge 12: Antalstabeller**
  Uafhængighedstest, homogenitetstest og Goodness-of-Fit test for kategorisk data.
* **Uge 13: Variansanalyse (ANOVA)**
  Sammenligning af middelværdier på tværs af flere grupper.
